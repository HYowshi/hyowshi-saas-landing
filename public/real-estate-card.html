Privacy Policy

1. Direct Log Processing: Content items submitted to the Concierge Form (Name, phone, specific queries) are securely routed to the agent's Google Spreadsheet database.

2. Encrypted Lines: All transmission channels utilize HTTPS SSL certificates preventing third-party wiretappings.

3. Erase Command: Users retain absolute authority to order complete metadata deletion of earlier messages anytime.

Google Script for Sheet Delivery:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheets();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([data.name, data.phone, data.email, data.message, data.date]);
  return ContentService.createTextOutput("Success")
    .setMimeType(ContentService.MimeType.TEXT);
}
